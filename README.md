# Minas Dev Website

## Tooling

- reeakt
- SSR (Server-Side Rendering)
- react
- react-router
- redux
- redux-saga
- immutable.js
- redux-immutable
- styled-components
- webpack
- jest & enzyme
- prettier
- ducks _(a modern and optimized way to work with redux structure)_
- axios
- express

## Installation

```
$ git clone https://github.com/minasdev/minasdev.org.git
$ npm install
```

## CLI

#### Run the Development Mode

```
$ npm run dev
```

#### Run the Production Mode

```
$ npm run start
```

#### Run the tests

```
$ npm run test
```
